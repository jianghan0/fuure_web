# fuure_web

## Enviroment
nodejs: 11.15.0  
Nuxt: 2.9.2  
typescript  

## Build Setup

``` bash
# set env file
$ cp .env.sample .env

# install dependencies
$ yarn install

# serve with hot reload at localhost:3000
$ yarn dev
```
For detailed explanation on how things work, check out [Nuxt.js docs](https://nuxtjs.org).
